# Ferramenta de predição

## Requisitos
- Python instalado
- Pip instalado

## Instalação

Em um terminal, no diretório (pasta) do programa, entre o comando

``` bash
pip install -r requirments.txt
```

## Execução

Em um terminal, no diretório (pasta) do programa, entre o comando

``` bash
streamlit run Streamlit.py
```
